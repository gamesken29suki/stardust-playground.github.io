## 🎮 サーバーの概要と活動内容

「星屑の遊び場」は、INFP-T的な心地よさを大切にした、多趣味なメンバーが集まるコミュニティです。

### 🕹️ メインコンテンツ

* **Gaming:** Minecraft (Hypixel)、Osu! などのゲームプレイ募集、情報交換。
* **Creative:** 東方Project、ブルーアーカイブ関連のファンアート、SS、BGM制作の共有と交流。
* **Harmony:** 競争を避け、お互いの価値観を尊重する平和な運営を心がけています。
